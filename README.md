# 🎓 Suivi Académique des Étudiants

Projet full-stack développé avec **Spring Boot** (backend) et **Angular** (frontend) pour assurer un **suivi académique efficace, moderne et transparent** des étudiants au sein d’un établissement universitaire marocain.

---

## 1. 📌 Contexte du Projet

Dans le contexte universitaire marocain, le **suivi académique des étudiants** est encore souvent **manuel**, **peu transparent** et **difficilement accessible**, tant pour les étudiants que pour les enseignants.

👉 Cela entraîne :
- Un **manque de visibilité** sur la performance des étudiants,
- Des **retards dans le traitement des résultats**,
- Des **difficultés à détecter à temps les étudiants en difficulté**.

---

## 2. 📚 Modalités Pédagogiques

Ce projet a une vocation **pédagogique** et permet de :

✅ Apprendre le **développement web full-stack** (Spring Boot + Angular)  
✅ Appliquer l’**architecture MVC** et les **bonnes pratiques** en génie logiciel  
✅ Mettre en œuvre des **API REST**, une **authentification sécurisée**, et la **gestion des rôles**  
✅ Gérer une **base de données relationnelle** (MySQL ou PostgreSQL)  
✅ Travailler sur un **projet concret et professionnalisant**

---

## 3. 👥 User Stories

### 🚀 Étudiants

- M’inscrire et me connecter pour accéder à mon espace personnel  
- Voir mon tableau de bord pour suivre ma progression  
- Consulter mes cours (modules suivis)  
- Voir mes notes par devoir et examen  
- Suivre mon taux de présence  
- Lire les feedbacks de mes enseignants

### 👨‍🏫 Enseignants

- Me connecter de manière sécurisée  
- Créer et gérer mes cours  
- Ajouter les étudiants à mes cours  
- Saisir les notes et les évaluations  
- Enregistrer les présences  
- Ajouter des feedbacks personnalisés

### 🏫 Administration

- Voir les statistiques globales par semestre  
- Détecter automatiquement les étudiants en difficulté  
- Exporter les données pour les rapports  
- Gérer les utilisateurs et les modules

---

## 4. 🛠️ Technologies Utilisées

### Backend
- **Java 17** / **Spring Boot**
- **Spring Data JPA**
- **Spring Security + JWT**
- **REST API**
- **MySQL** / **PostgreSQL**

### Frontend
- **Angular 18+**
- **Angular Router & Forms**
- **TailwindCSS** ou **Bootstrap**
- **RxJS / HttpClient**
- **JWT Interceptor**

### Outils & Autres
- **Maven** / **npm**
- **Postman** pour tester les endpoints
- **Git** / **GitHub**
- **Figma** (pour les maquettes UI)
- **IntelliJ / VS Code**
